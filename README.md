#Remodel Mate Client

##Installation 
1. `$ npm i`

*Development:*

2. `$ npm run fake-server` - Will run a mock server for job data 
3. `$ npm run dev`

*Production:*

4. `$ npm run prod`

5. Open browser to: `localhost:3001`
5. Navigate to: `localhost:3001/jobs` to get job data
